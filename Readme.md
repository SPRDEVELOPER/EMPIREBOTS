# 👑 EMPIRE MUSIC BOT

A powerful Telegram Music Bot with Voice Chat support, built for groups.

## Features
- 🎵 Play music in Voice Chats
- 📺 YouTube, Spotify, SoundCloud, Apple Music, Resso support
- 🎬 Live streaming
- 🔁 Loop, shuffle, seek support
- 🛡️ Admin management tools
- 🌍 Multi-language support
- 24×7 music playback

## Setup

### Requirements
- Python 3.10+
- MongoDB URI
- Telegram API credentials
- Session string(s)

### Environment Variables
Copy `sample.env` to `.env` and fill in your values:

```
API_ID=
API_HASH=
BOT_TOKEN=
OWNER_ID=
LOGGER_ID=
STRING_SESSION=
MONGO_DB_URI=
COOKIE_URL=
```

### Deploy

**Heroku:**
[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)

**Local:**
```bash
pip install -r requirements.txt
python -m EmpireMusic
```
